amidi -l
damit die hw-id von Emphase bestimmen, zB 0, dann

amidi -p hw:0 -S "<hex data>", zb "90 3C 01" (== NoteOn Key Velocity)

Testmodus derzeit :
	- "90 3C 01" (Note On) == Audio Engine ON (single) 		==> LED "LB" ON
	- "80 3C 01" (Note Off) == Audio Engine OFF (single) 		==> LED "LB" OFF
	- "A0 3C 01" (Aftertouch) == Audio Engine Heartbeat (single)	==> LED "LB" toggling

Heartbeat schicken:
while true; do amidi -p hw:0 -S "A0 3C 01"; sleep 0.05; done
